# Text Encoding API
: 문자열 인코딩과 디코딩을 위한 인터페이스 집합  
: 문자열을 바이너리 데이터로 인코딩하거나, 바이너리 데이터를 문자열로 디코딩함  

**스펙**  
https://encoding.spec.whatwg.org


**인터페이스**
- TextEncoder
- TextDecoder
- TextEncoderStream
- TextDecoderStream
