# Ledger System
: 경제적 가치를 추적하고 기록하는 데이터 구조와 프로세스의 집합  
: 금융 거래, 자산 이전, 소유권 변경 등 경제적 활동을 체계적으로 기록하고 관리하는 통합 시스템  
: 원장 시스템은 단순한 기록이 아닌 거래의 정확성과 신뢰성을 보장하는 메커니즘을 포함함  

- 원장 개념과 원리
- 원장 시스템 기능
- 원장 시스템 분류
- 디지털 화폐와 원장
- 원장 시스템 응용 분야


**특징**
- 신뢰성 : 데이터의 정확성과 일관성을 보장함
- 보안성 : 무단 접근이나 조작을 방지하고 데이터 무결성을 보장함
- 투명성 : 권한에 따라 거래 내역에 대한 가시성을 제공함
- 모든 거래 내역을 시간순으로 추적하고 검증함
