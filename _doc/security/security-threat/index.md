# Security Threat
: 자산에 손상을 발생시킬 수 있는 잠재적 가능성을 지닌 위협 요소들  
: 인적, 기술적, 환경적 등 다양한 위협 요인이 있으며 고의적이거나 비고의적일 수 있음  
: 자연 재해는 예기치 않은 비의도적 위협으로 물리적 위협에 해당함  

**위협 vs 취약점 vs 공격**  
: 위협은 잠재적으로 해를 끼치는 가능성과 상황을 포함한 실현되지 않은 상태  
: 위협이 악용할 수 있는 시스템, 애플리케이션, 프로세스 등의 약점이나 결함  
: 공격은 위협이 실제로 실행되어 자산에 손상을 발생시키는 구체적 행위  

공격 | 설명
---|---
소극적 공격 | 자산을 손상 시키지 않으면서 정보를 탐지하거나 수집함
적극적 공격 | 자산을 직접적으로 변경하여 시스템에 영향을 미치는 공격


**위협 유형**

유형 | 설명 | 영향
---|---|---
가로채기 | 권한없는 주체가 불법적으로 자산에 접근하고 데이터를 탈취하는 행위 | 기밀성
가로막음 | 권한있는 주체가 리소스에 접근하지 못하도록 방해하는 행위 | 가용성
변조 | 데이터를 무단으로 변경하거나 조작하여 원본 정보를 왜곡하는 행위 | 무결성
위조 | 허위 데이터를 생성하거나 신원을 속여 시스템을 기만하는 행위 | 무결성

## Threat Source
: 보안 위협의 근본 원인 또는 위협이 발생할 수 있는 출처

- 자연적 위협 : 지진, 홍수, 화재, 폭풍 등
- 환경적 위협 : 전력 공급 장애, 냉각 시스템 오류, 통신망 불안정
- 기술적 위협 : 시스템 오류, 하드웨어 고장, 네트워크 결함, 소프트웨어 취약점
- 인적 위협
  - 고의적 : 공격, 정보 유출
  - 비고의적 : 인적 실수, 정책/절차 실수, 보안 인식 부족



## Threat Actors
: 보안 위협을 실행하거나 실행할 수 있는 주체들  

**외부 행위자**
- 사이버 범죄자
- 해커, 해킹 그룹
- 국가 지원 공격자 : 국가가 후원하거나 지휘하는 해커나 해킹 그룹
- 핵티비스트 : 정치적·사회적 목적을 위해 해킹을 수행하는 사람이나 그룹
- 산업 스파이
- 사이버 테러리스트

**내부 행위자**
- 악의적 내부자 -- 고의적
- 부주의한 직원 -- 비고의적 (사고, 실수)
- 전직 직원
- 파트너사
- 제3자 계약자



### APT
Advanced Persistent Threat  
: 지속적 위협이란 장기간에 걸쳐 지속적으로 특정 대상을 공격하는 고도화된 위협 행위자  
: 위협 행위자 중에서도 지능적이고, 조직적으로, 장기간에 걸쳐 지속적으로 목표를 공격함  

**특징**
- 정교하고 고도화된 기술 사용
- 공격이 장기간 지속됨
- 무작위 공격이 아니라 특정 목표가 존재함
