# OS Script
: 운영체제의 [쉘](../../os/os-component.md#shell) 환경에서 실행되는 명령어 집합  
: 운영체제와 상호작용하며 명령어를 순차적으로 실행하며 일반적으로 인터프리터 방식으로 동작됨  
: 운영체제마다 다른 쉘을 사용하여 스크립트 문법과 세부 기능은 다를 수 있음  
: 시스템 관리, 배치 작업, 특정 작업 자동화, 구성 관리 등을 위해 사용됨  

- [Bash](./bash/index.md)
- [PowerShell](./powershell/index.md)


환경 | 쉘 | 용어
---|---|---
Linux   | Bash | 쉘 스크립트
Windows | CMD  | 배치 스크립트
Windows | PowerShell | PowerShell 스크립트


**쉘**  
: 명령을 해석하여 커널에 전달하는 프로그램 (명령어 인터프리터)  

**쉘 환경**  
: 사용자와 운영체제가 상호작용하는 인터페이스를 제공하는 환경  
: 쉘, 환경 변수, 설정 파일, 별칭 등 다양한 요소를 포함해 실행하는 환경
