# Security Strategy
: 보안 전략이란 조직의 자산을 보호하기 위한 포괄적인 계획을 의미함  
: 조직이 보호할 자산을 식별하고 이를 보호하기 위한 장기적인 청사진과 로드맵을 제시함  
: 보안 위협을 효과적으로 관리/대응하기 위한 방향성, 원칙, 우선 순위 등 실행 방법을 포함함

**보안 전략 목적**
- 위협으로부터 조직의 자산을 보호함
- 조직의 비즈니스 연속성을 보장함
- 보안 관련 법규를 준수함
- 보안을 통해 고객으로부터 신뢰를 확보함
