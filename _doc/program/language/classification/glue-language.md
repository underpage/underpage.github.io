# Glue Language
: 서로 다른 소프트웨어 구성 요소를 연결하고 통합하는데 사용되는 프로그래밍 언어  
: 일반적으로 높은 수준의 추상화를 제공하여 복잡한 작업을 간단하게 처리 가능함  
: 접착 언어들은 공통적으로 동적 타이핑 언어와 인터프리터 언어 특징을 가짐  
: 시스템 간 데이터 변환, 파이프라인 구축, 자동화 처리에 특화됨  

**사용 사례**
- API 통합
- 레거시 시스템과 현대적 시스템 간의 브릿지
- 데이터 변환 및 처리
- 시스템 관리 및 자동화
- 테스트 자동화
- 로그 분석


**언어 종류**
- Perl
- Lua
- Ruby
- Python
- JavaScript
- Groovy
