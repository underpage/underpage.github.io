# Solidity Modifier
≒ Specifier  
: 변수, 함수, 생성자의 동작을 변경하거나 제한하는 역할을 하는 키워드  
: 공식 문서에서는 이를 지정자로 명시하고 사용자가 modifier 키워드로 정의한 코드 블럭을 수정자라고 구분함  
: 지정자와 수정자 모두 동작이나 특성을 정의하거나 제한하므로 넓은 의미에서는 수정자로 표현하기도 함

- 내장 수정자
  - 접근 제어 지정자
  - 상태 변경 지정자
- 사용자 정의 수정자


**수정자 vs 지정자**

구분 | 수정자 | 지정자
---|---|---
정의 | 함수 실행 전이나 후에 특별한 로직을 추가함 | 변수나 함수의 속성을 정의하는 키워드
용도 | 함수의 실행 조건이나 흐름을 제어함 | 접근 제어, 상태 변경 등 메타데이터를 부여함
정의 방식 | 사용자가 수정자를 직접 정의함 | 언어 수준에서 내장 키워드로 제공됨
