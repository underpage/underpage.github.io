# AI History
: 1943년 워렌 맥컬럭과 월터 피츠가 생물학적 뉴런을 모방한 수학적 모델에 대한 논문을 발표함  
: 이는 뉴런의 작동 원리를 이진 논리로 표현한 계산 모델로 인간의 사고나 인지를 수학적으로 설명함  
: 이 논문은 현대 인공지능의 이론적 토대를 마련하고 발전 방향에 결정적 영향을 미침  

**아이디어**  
: 하나의 뉴런은 단순한 이진 논리 단위로 작동하고 다수의 뉴런들이 연결되어 복잡한 논리 회로를 구성함

**맥컬럭-피츠 모델**
1. 입력 처리 : 뉴런은 여러 입력 신호를 받음
2. 임계값 연산 : 입력 신호들의 합이 일정 임계값을 넘으면 1(활성화) 아니면 0(비활성화)을 출력
3. 논리 연산 : 이 뉴련들을 연결하여 논리 연산을 구현할 수 있음


**발전 타임라인**
```
1956년  존 매카시가 컴퓨터가 사람처럼 사고하고 학습하는 개념 제안 (인공 지능)
1959년  아서 새뮤얼이 컴퓨터가 명시적인 프로그래밍 없이 스스로 학습하는 개념 정의 (머신 러닝)
2006년  제프리 힌튼이 여러 층의 심층 신경망을 사용하는 기법과 용어 정의 (딥러닝)


1950년대  최초로 인공 신경망 장치 개발 (SNARC)
          데이터로부터 학습하기 위한 신경망 모델 개발 (퍼셉트론)
        
1960년대  서로 다른 종류의 데이터를 구분하는 경계선을 의미하는 개념 등장 (결정 경계)
          컴퓨터가 규칙적인 형태나 특징을 찾아내는 패턴 인식 기술 발전
          규칙을 기반으로 동작하는 최초의 챗봇 등장 (ELIZA, Shakey)

1970년대  컴퓨터 하드웨어 기술이 발전하며 복잡한 수학 계산이 가능해짐  
          신경망 가중치를 효율적으로 학습시키는 역전파 알고리즘 이론 개발
          데이터를 K개 그룹으로 나누는 클러스터링 알고리즘 개발

1980년대  정답을 알려주면서 컴퓨터를 훈련하는 학습 방법론이 체계화됨 (지도 학습)
          역전파 알고리즘을 신경망에 적용해 다층 신경망 훈련을 가능하게 함 (다층 퍼셉트론)
          시퀀스 데이터 처리를 위한 순환 신경망 구조 개발 (RNN)
          질문을 단계별로 던져 최종 결론에 도달하는 결정 트리 알고리즘 개발

1990년대  정답을 알려주지 않고 컴퓨터가 스스로 패턴을 찾는 학습 방법론이 체계화됨 (비지도 학습)
          데이터를 분류하기 위해 최적의 경계선을 찾는 서포트 벡터 알고리즘 개발
          여러 개의 결정 트리를 결합한 랜덤 포레스트 알고리즘 개발
          역전파 알고리즘을 지도 학습에 적용한 합성곱 신경망 구조 개발 (CNN)
          RNN의 장기 의존성 문제를 해결한 개선된 순환 신경망 구조 개발 (LSTM)

2000년대  환경과 상호작용하면서 보상을 통해 학습하는 방법론 등장 (강화 학습)
          약한 학습기를 결합해 강한 학습기를 만드는 부스팅 알고리즘이 발전함
          딥 볼츠만 머신, 딥러닝 개념을 제정립함

2010년대  여러 층의 합성곱 신경망을 깊게 쌓은 구조인 AlexNet 모델 등장
          서로 경쟁하며 학습하는 두 개의 신경망으로 새로운 데이터를 생성하는 기법 개발 (GAN)
          순환 신경망 없이 어텐션 메커니즘만으로 언어를 처리하는 Transformer 모델 개발
          양방향으로 문맥을 이해하는 BERT 모델 등장
          대규모 데이터를 기반으로 텍스트를 생성하는 GPT-2 모델 등장

2020년대  대화형 생성 AI 서비스인 ChatGPT가 등장하며 생성형 AI가 대중화 됨
          구글 Bard, 마이크로소프트 Bing Chat 등 대화형 AI 기술 등장
          LLM 한계를 극복하기 위해 외부 지식을 검색하고 결합하는 RAG 기술 등장
          텍스트, 이미지, 음성을 동시에 처리하는 멀티모달 기술 등장
          자율적으로 목표를 달성하기 위해 도구를 사용하고 환경과 상호작용하는 AI 에이전트 시스템 등장
          자연어 명령을 코드로 변환하는 코드 생성 AI 서비스 등장
```


**발전 단계**  
규칙 기반 시스템(기호 주의) => 머신 러닝(일부 기호 주의 + 연결 주의) => 딥러닝 (연결 주의)
