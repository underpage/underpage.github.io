# Communication Types
: 통신은 정보의 형태, 기술 특성, 전달 매체 등 다양한 기준에 따라 분류됨  


**발전 단계**
- 전자 통신 : 전신, 전화 등
- 방송 통신 : 라디오, 텔레비전 (단방향 통신)
- 정보 통신 : 디지털 기술 기반 (양방향 통신)

**전송 기술**
- 아날로그 통신
- 디지털 통신

**전송 매체**
- 유선 통신
- 무선 통신
- 광통신

**정보 형태**
- 음성 통신
- 데이터 통신
- 영상 통신
- 멀티미디어 통신

**전송 범위**
- 근거리 통신
- 광역 통신
- 이동 통신

**연결 형태**
- 점대점 통신 : 두 지점이 직접 연결하는 통신 방식
- 다지점 통신 : 여러 지점이 하나의 매체를 공유하는 통신 방식
