# Solidity Variable
: 스마트 컨트랙트에서 데이터를 저장하고 참조하기 위해 사용되는 식별자  
: 솔리디티 변수는 선언 위치와 사용 목적에 따라 분류됨  
: 상태 변수는 디스크에 저장되지 않고 블록체인 저장소에 영구 저장됨  
: 전역 변수는 솔리디티에서 제공하는 일종의 환경 변수로 사용만 가능하며 정의는 불가능함
