# Spring Web Flow
: 상태 기반 웹 애플리케이션의 데이터 흐름을 제어하기 위한 프레임워크  
: 웹 애플리케이션의 데이터 흐름과 화면 전환을 체계적으로 관리할 수 있도록 도와줌  
: 회원 가입, 주문 결제 등 여러 단계를 거치는 프로세스에서 데이터를 유지하기 위해 사용됨  

**상태 기반 시스템** (Stateful System)  
: 상호작용 과정에서 이전 데이터를 기억하고 유지하는 시스템  
: 예를 들어 브라우저에서 새로고침을 하거나 일정 시간이 지나도 이전 데이터가 유지됨  

**주요 용어**

용어 | 설명
---|---
흐름 | 사용자가 수행하는 단계적인 작업 흐름을 정의하는 요소
상태 | 특정 시점의 애플리케이션의 상태
전이 | 하나의 상태에서 이벤트에 의해 다른 상태로 전환되는 과정
스코프 | 흐름 내에서 데이터를 저장하고 관리하는 방식
이벤트 | 사용자 액션이나 시스템 의해 발생하는 액션으로 상태 전이의 트리거 역할을 함
