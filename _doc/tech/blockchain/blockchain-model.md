# Blockchain Model
: 접근 권한, 참여 범위, 거버넌스 구조에 따라 분류되는 블록체인의 운영 방식과 특성을 정의하는 분류 체계  

- [퍼블릭 블록체인](#public-blockchain)
- [프라이빗 블록체인](#private-blockchain)
- [컨소시엄 블록체인](#consortium-blockchain)
- [하이브리드 블록체인](#hybrid-blockchain)



## Public Blockchain
: 누구나 참여하고 검증 가능한 개방형 네트워크  

- 비트코인
- 이더리움
- 솔라나
- 폴리곤
- 아발란체 



## Private Blockchain
: 특정 조직이 허가한 사용자만 접근 가능한 폐쇄형 네트워크  
: 네트워크를 통제해 일반 사용자는 블록체인 데이터를 확인할 수 없음  

- Hyperledger Fabric
- Corda



## Consortium Blockchain
: 여러 조직이 공동으로 운영하는 반개방형 네트워크  
: 조직의 허락이 필요하지만 사용자의 신원을 밝히지 않음  

- Quorum
- BFT



## Hybrid Blockchain
: 퍼블릭과 프라이빗의 장점을 조합한 네트워크

- XinFin
