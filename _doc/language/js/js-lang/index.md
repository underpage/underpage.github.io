# JavaScript Programming Language
: 다중 패러다임 언어 (명령형, 함수형, 객체지향)  
: 일급 함수, 고차 함수, 순수 함수를 지원하여 함수형 프로그래밍이 가능하며 일부 메소드를 통해 선언형 프로그래밍을 지원함  
: 프로토타입 기반 객체지향 언어로 프로토타입을 통해 객체 생성과 상속이 이루어짐  
: 데이터 타입을 명시적으로 선언하지 않는 동적 타입 언어로 런타임에 객체 동적 접근과 변경이 용이함

**영향받은 언어**

언어 | 영향
---|---
Lisp    | 변수 스코프
C, Java | 연산자, 표현식, 데이터 타입, 객체 관계
Self    | 프로토타입 기반 객체지향, 프로토타입 상속(위임)
AWK     | 속성 반복, 함수 선언
HyperTalk    | 이벤트 핸들링
Lisp, Scheme | 일급 함수, 클로저  
Peal, Python | 문자열, 배열, 정규 표현식
