# Communication
: 통신이란 두 개 이상 개체가 정보를 전달하거나 교환하는 행위를 의미  
: 통신은 정보를 주고 받는 것만 아니라 상호작용을 통해 의미가 형성되는 과정을 포함함  
: 정보는 언어적, 시각적, 청각적 등 여러 형식으로 표현되며 다양한 매개체를 통해 전달됨  
: 몸짓, 문자, 신호는 통신 수단의 일종이며 시대에 따라 기술이 발전하며 통신 수단과 방식도 진화함  
: 전화기, 컴퓨터, 핸드폰과 같은 새로운 통신 수단이 등장하면서 이를 위한 정보 통신 기술이 함께 발전함  
: 현대의 통신 수단은 정보를 디지털화하여 더욱 빠르고 정확하게 전달할 수 있음
