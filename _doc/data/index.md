# Data
: 데이터의 어원은 라틴어 dare(give) => datum(thing given)에서 유래  
: 데이터 정의는 문화적 배경이나 수학, 철학, 과학 등 분야에 따라 다양하게 해석됨  
: 현대 사회에서 데이터는 디지털 기술과 컴퓨터 시스템을 통해 대규모로 생성, 수집, 활용됨
