# Program Optimization
: 프로그램 성능을 향상시키기 위해 실행 속도, 메모리 사용량, 에너지 효율 등을 개선하는 기법  
: 컴파일러를 통해 런타임 환경에서 자동으로 적용되거나 사용자가 직접 코드 수준에서 적용함  
: 최적화 효과는 실행 환경, 하드웨어 사양, 워크로드(입력 데이터)에 따라 달라질 수 있음

**최적화 목표**

유형 | 설명
---|---
시간 최적화 | 프로그램 실행 시간을 빠르게 하고 지연 시간을 감소하여 빠른 응답성을 확보함
공간 최적화 | 메모리 사용량을 최소화해 자원을 효율적으로 관리함
계산 최적화 | 불필요한 연산 제거나 중복 계산을 방지하여 연산 비용을 최소화함
코드 최적화 | 코드 가독성, 유지보수성, 실행 효율성, 구조적 품질을 개선함
실행 환경 최적화 | 하드웨어와 소프트웨어 환경에 따라 성능을 개선해 자원 낭비를 막음
네트워크 최적화 | 데이터 전송 속도 향상, 대역폭 절감, 통신 지연 최소화 등 통신 효율성을 개선함
확장성 최적화 | 시스템 규모 확장시 성능을 유지함
