# Visual Programming Language
: 그래픽 요소와 시각적 표현을 사용하여 프로그램을 생성하는 프로그래밍 언어 또는 환경  
: 프로그램 요소를 시각적으로 표현하여 다이어그램, 아이콘 등을 시각적으로 배치하여 프로그램을 구성함  
: 개발 지식이 없어도 접근하기 쉬워 교육 분야에서 프로그래밍 개념을 학습하기 위해 사용됨  
: 복잡한 대규모 시스템보다는 특정 도메인이나 목적에 맞는 애플리케이션 개발에 적합함  

**유형**
- 블록 기반
- 아이콘 기반
- 다이어그램 기반
- 노 코드
- 로우 코드



## Block-Based
: 미리 정의된 블록을 조합하여 프로그램을 만드는 방식  
: 일반적으로 교육용 프로그래밍 환경에 사용됨  

- Scratch
- Blockly



## Low-Code
: 시각적 인터페이스를 주로 사용하나 필요에 따라 일부 코딩이 가능한 시각적 프로그래밍의 발전 형태  
: 필요에 따라 시각적 개발과 코드 작성을 자유롭게 전환하여 복잡한 기능 구현이 가능함  
: 모든 로우 코드가 시각적 프로그래밍인 것은 아니고 일부 로우 코드가 시각적 프로그래밍에 포함됨  

- Microsoft Power Platform
- OutSystems
