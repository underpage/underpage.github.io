# Cryptography
: 암호학은 정보를 보호하기 위한 수학적 기법과 원리를 연구하는 학문  
: 데이터를 암호화하고 복호화하는 암호 기술과 이를 해독하고 분석하는 암호 해독(분석)을 다룸  

- [암호 기술](./cryptographic-technology.md)
- [암호 분석](./cryptanalysis.md)
- [암호 평가](./cryptographic-evaluation.md)


**암호**  
: 정보의 기밀성을 보장하기 위해 특정한 규칙과 키를 사용하여 데이터를 변환하는 기술  


**목표**  
- 기밀성 : 허가받지 않은 개체가 데이터를 이해할 수 없도록 함
- 무결성 : 데이터가 변조되거나 손상되지 않았음을 보장함
- 인증 : 통신하는 개체의 신원과 데이터의 출처를 검증함
- 부인 방지 : 송신자가 메시지 전송을 부인할 수 없도록 보장함


**주요 용어**

용어 | 설명
---|---
평문 | 암호화되지 않은 원본 메시지
암호화 | 평문을 특정 규칙(알고리즘)과 키를 통해 암호문으로 변환하는 과정
암호문 | 암호화를 통해 코드화된 데이터로 원래 내용을 알 수 없음
복호화 | 암호문을 키를 사용해 원래의 평문으로 복원하는 과정
키 | 암호화와 복호화에 사용되는 값


**암호 기법**  
: 치환과 전치는 고전 암호에서 시작되어 현대 암호 알고리즘의 기본 원리로 발전함  

용어 | 설명
---|---
치환 | 문자를 다른 문자나 기호로 대체
전치 | 문자의 순서를 변경



## Cryptography History
: 암호학은 기술이 발전함에 따라 다양한 방식으로 발전함  
: 초기에는 인간이 직접 문자를 변환했으나 현대에는 기계와 수학적 알고리즘을 활용함  


**고대 암호**  
: 인간이 직접 문자를 대체하거나 순서를 바꾸는 방식으로 암호화가 이루어짐  
: 키 규칙이 단순하여 분석을 통해 쉽게 해독 가능함  

치환 암호 | 설명
---|---
시저 암호 | 알파벳을 일정 간격으로 이동하여 변환함
모노알파베틱 치환 암호 | 한 문자를 다른 문자로 대체함
비즈네르 암호 | 여러 개의 시저 암호를 조합해 다중 알파벳으로 치환함

전치 암호 | 설명
---|---
스키테일 암호 | 막대기에 종이를 감아 문자를 재배열함
열 전치 암호 | 문자의 열 순서를 변경함
행 전치 암호 | 문자의 행 순서를 변경함


**근대 암호**  
: 기계 장치를 이용해 암호 변환의 복잡성을 높이고 자동화함  
: 기계의 설정 값을 조합해 키를 생성해 보안이 강화되었으나 사전에 키를 공유해야 하는 문제가 존재함  

기계식 암호 | 설명
---|---
에니그마 암호 | 회전자를 이용한 복잡한 치환 방식
로터 머신 암호 | 다중 회전 메커니즘을 사용하여 암호화
폴리비우스 광장 | 문자를 좌표 기반으로 암호화


**현대 암호**  
: 컴퓨터를 기반으로 수학적 알고리즘을 적용해 복잡한 계산 과정을 거쳐 암호화가 이루어짐  
: 컴퓨터 연산을 통해 키를 생성하고 관리하며 공개키 암호 기술을 도입해 키 분배 문제를 해결함  
: 키의 길이가 길고 무작위로 생성되며 주기적으로 갱신하여 보안성이 극대화됨
