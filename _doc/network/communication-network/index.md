# Communication Network
: [통신](../communication/index.md)을 위해 장치, 기술, 서비스가 결합된 구체적인 구현체  
: 효율적이고 신뢰성있는 정보 전송과 자원 공유를 목표로 하며 이를 위해 다양한 기술을 활용함  

- [통신망 종류](./communication-type.md)
- [통신망 주소 체계](./communication-addressing.md)
- [통신망 기술](./communication-technology.md)
- [통신망 성능](./communication-performance.md)


**network vs communication network**  
: [네트워크](../network.md)는 정보 전송을 위한 구조와 기술을 포괄하는 추상적인 개념  
: 통신망은 이러한 네트워크를 기반으로 정보 전송을 위한 구체적인 시스템  
: 현대에는 네트워크와 통신망의 경계가 모호하며 대부분 네트워크는 통신 기능을 포함함  

구분 | 네트워크 | 통신망
---|---|---
범위 | 추상적인 개념 | 정보 전송을 목적으로 하는 특정 네트워크
구성 요소 | 물리적 논리적 장치와 프로토콜 | 네트워크 구성 요소와 통신 기술
예시 | LAN, WAN | 전화망, 이동통신망


**history**
```
1840년대  모스 부호가 개발되며 최초로 전기 통신이 가능해짐
1870년대  전화가 발명되고 유선 전화망이 형성되어 실시간 음성 통신이 가능해짐

1900년대  라디오가 개발되며 음성 데이터의 무선 전송이 가능해짐
1940년대  전자식 컴퓨터가 발명되고 디지털 데이터 통신의 기초가 마련됨
1960년대  위성 통신이 개발되며 국제 통신망 기반이 마련됨
1970년대  ARPANET과 패킷 교환 방식이 도입되어 인터넷 기반이 마련됨

1970년대  디지털 통신으로 전환되어 데이터 통신의 효율성과 신뢰성이 향상됨
1980년대  이동통신망 도입되며 무선 음성 통신이 대중화됨
1990년대  2세대 이동통신망이 도입되고 인터넷이 대중화되어 데이터 통신이 크게 성장함
2000년대  3세대와 4세대 이동통신이 도입되고 모바일 데이터 통신이 고도화되고 광대역 인터넷이 보급됨
2010년대  5세대 이동통신과 사물인터넷이 등장하며 초연결 시대가 시작됨
```
