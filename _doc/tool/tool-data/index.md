# Data Engineering Tool
: 데이터의 수집, 처리, 변환, 저장, 관리 등을 위한 도구 및 프레임워크



## Data Processing Framework
: 대용량의 데이터를 효율적으로 처리하기 위한 시스템

**Batch Processing**
- Apache Hadoop
- Apache Spark

**Stream Processing**
- Apache Kafka
- Apache Storm
- Apache Samza
- Apache Pulsar

**Unified Processing**
- Apache Spark
- Apache Flink
- Apache Beam



## Data Integration Tool
: 여러 소스의 데이터를 결합하고 통합하는 도구

**ETL/ELT**
- Apache NiFi
- Talend Open Studio
- Pentaho Data Integration
- dbt
- Airbyte



## Data Pipeline Management Tool
: 데이터 흐름 프로세스 설계 및 관리를 위한 도구

**Data Orchestration**
- Luigi
- Apache Airflow
- Prefect
- Dagster

**Event-Driven Processing**
- Apache Flink
- Materialize

**Message Broker**
- RabbitMQ
- Apache Kafka
- Apache Pulsar



## Data Governance & Catalog
: 데이터 자산 관리, 품질, 접근성 향상을 위한 도구  

**Data Discovery**  
: 조직 내 데이터 자산을 발견, 문서화, 카탈로그화하는 도구

- Apache Atlas
- Amundsen
- DataHub

**Data Lineage**
- OpenLineage
- Marquez
