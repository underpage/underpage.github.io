# Compression Streams API
: 데이터 압축과 해제를 스트리밍 방식으로 처리하기 위한 인터페이스 집합  
: gzip 또는 deflate 형식을 사용하여 데이터 스트림의 압축 및 해제를 지원함  
: 대용량 데이터를 웹 브라우저 내에서 실시간으로 처리하기 위해 사용됨  

**스펙**
https://compression.spec.whatwg.org/
