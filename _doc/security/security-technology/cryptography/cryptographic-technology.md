19세기 이후 정치적, 군사적 정보 보호의 중요성이 커지면서 암호의 실용성이 강조되었고
20세기 세계대전으로 기계식 암호 장치가 등장하면서 암호 분석과 해독 기술이 크게 발전함

이후 컴퓨터가 등장하고 정보가 디지털화 되면서 본격적으로 암호학이 발전함

**암호 기법**  
: 대치와 전치는 고전 암호의 주요 기법으로 현대 암호 알고리즘의 기본 원리로 발전함  

기법 | 설명
---|---
대치 (치환) | 데이터의 요소를 다른 값으로 대체하여 의미를 변형함
전치 (전위) | 데이터의 비트나 바이트 순서를 변경하여 원래 형태를 숨김
XOR 연산 | 키를 사용해 평문을 배타적 논리합을 수행해 암호화함


**암호 기술의 기본 원칙**
- 커크호프의 원칙
  - 암호 시스템의 보안은 키의 비밀성에 의존해야 함
  - 알고리즘 자체는 공개되어도 안전해야 함
- 섀넌의 혼돈과 확산 : 암호 알고리즘은 혼돈과 확산의 성질을 가져야 함
- 개방형 검토
  - 암호 알고리즘은 공개적으로 검토되고 분석될 수 있어야 함
  - 다수의 전문가를 통해 검증되어 안정성을 확보해야 함
- 최소 권한 : 암호 시스템은 필요한 최소한의 권한만 부여해야 함
- 심층 방어 : 여러 층의 보안 메커니즘을 통해 전체적인 시스템 보안을 강화해야 함



### 클로드 섀넌
: 통신의 수학적 이론(1948)과 암호 시스템의 통신 이론(1949)을 발표해 암호 기술의 이론적 토대를 마련  
: 정보 이론을 암호학에 적용하여 완벽한 비밀성 개념과 조건을 수학적으로 정립함  
: 혼돈과 확산이라는 암호 설계의 두 가지 기본 원칙을 제시함  
: 원타임 패드(OTP)가 수학적으로 무조건적 보안을 제공하는 암호 시스템임을 증명함  

- 혼돈 : 암호문과 키의 관계를 복잡하게 만들어 키를 추측하지 못하게 함 (대치 적용)
- 확산 : 암호문에서 평문의 특정 패턴이 드러나지 않도록 함 (전치 적용)
 

**통신의 수학적 이론**
- 정보량을 엔트로피로 수치화하여 정보를 정량적으로 표현함
- 비트 개념을 도입해 정보의 기본 단위를 정의함

**암호 시스템의 통신 이론**
- 암호학을 수학적으로 분석한 최조의 체계적인 이론
- 완전 비밀성 개념을 수학적으로 정의함
- 암호 시스템의 보안성을 정량적으로 평가하는 방법론을 제시함

**Perfect Secrecy**  
: 암호문을 관찰해도 평문에 대한 정보를 얻을 수 없는 상태  

- 키 공간의 크기가 평문 공간의 크기보다 커야함
- 키는 무작위적으로 생성되고 균등한 확률로 선택되어야 함
- 평문과 키의 조합은 유일한 암호문을 생성해야 함

**One-Time Pad**  
: 무작위 키를 한 번만 사용하여 평문과 XOR 연산을 수행하는 암호화 방식  
: 클로드 섀넌에 의하면 OTP는 완전 비밀성 조건을 만족하는 유일한 암호 방식임
