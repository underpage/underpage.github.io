# AI
Artificial Intelligence  
: 인간의 인지 능력(학습, 추론, 지각, 자연어 이해 등)을 컴퓨터 시스템으로 구현하는 기술  
: 지능적 행동을 모방하고 이를 확장하여 복잡한 문제를 자동화된 방식으로 해결하는 것을 목표로 함  
: 대량의 데이터로부터 패턴을 학습하고 이를 바탕으로 분류, 예측, 의사결정을 수행하는 방식으로 작동함  
: 인공지능은 논리학, 확률론, 통계학, 최적화 이론, 정보 이론 등 수학적 이론을 기반으로 함

**목표**
- 대규모 데이터를 빠르고 정확하게 분석하고 처리함
- 데이터의 숨겨진 패턴을 파악하고 새로운 인사이트를 도출함
- 인간의 개입 없이 의사결정을 수행하고 인간의 인지적 한계를 보완하고 확장함
- 새로운 데이터를 지속적으로 학습하고 변화하는 상황과 환경에 적응함
