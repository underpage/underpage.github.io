# Access Control
: 접근 제어란 자산에 대한 접근을 제어하는 보안 기술  
: 권한을 가진 주체만 특정 데이터, 시스템, 서비스에 접근할 수 있도록 통제하는 체계  
: 이를 통해 무단 접근을 방지하고 보안 정책 및 규정 준수를 보장함

**접근 제어 단계**  
: 일반적으로 접근 제어는 식별-인증-인가 단계를 거쳐 수행됨  
 
용어 | 설명
---|---
객체 | 접근 대상이 되는 데이터, 파일, 시스템, 서비스 등
주체 | 접근을 요청하는 사용자, 메시지, 시스템, 서비스 등
식별 | 주체의 신원을 확인하는 과정
인증 | 주체가 주장하는 신원을 증명하는 과정
인가 | 인증된 주체가 특정 자원에 접근하도록 권한을 부여하는 과정
