# Data Processing Framework
: 데이터 처리를 위한 구체적인 기술과 도구 집합  

**처리 방식**
- 배치 처리 프레임워크
  - Apache Hadoop
- 스트림 처리 프레임워크
  - Apache Kafka
  - Apache Storm
  - Apache Samza
  - Apache Pulsar
- 통합 처리 프레임워크
  - Apache Spark
  - Apache Flink
  - Apache Beam


**용도**
- ETL/ELT 도구
  - Apache NiFi
  - Talend
  - Apache Airflow
- 데이터 쿼리 및 OLAP
  - Apache Hive
  - Apache Drill
  - Presto/Trino
- 실시간 분석 및 OLAP
  - Apache Druid
  - ClickHouse
