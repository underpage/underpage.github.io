# Modern Software Engineering Practice
: 소프트웨어의 개발, 배포, 운영을 효율적으로 수행하기 위한 현대적 방법론과 기술 체계  
: 개발과 운영의 통합, 보안 강화, 신뢰성 보장 등 소프트웨어 엔지니어링 전반의 실천 방식을 포함함  
: 소프트웨어의 품질, 안정성, 확장성을 높이고 개발 및 배포 프로세스를 더욱 효율적으로 만드는 데 중점을 둠  

- DevOps
- DevSecOps
- SRE
- Chaos Engineering
- Platform Engineering
