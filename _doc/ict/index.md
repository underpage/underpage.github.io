# ICT
Information and Communication Technology  
: 정보 통신 기술  
: 정보의 생성, 저장, 처리, 전송을 위한 컴퓨팅 기술과 통신 기술의 융합  
: 컴퓨터, 소프트웨어, 네트워크, 센서 등을 활용하여 정보를 수집, 저장, 처리, 분석, 전달하는 기술 체계  
: 이러한 기술만 아니라 이를 활용하는 정보 서비스, 산업, 사회 패러다임을 포괄하는 개념  

**Computing vs ICT**  
: [컴퓨팅](../computing/index.md)은 주로 문제 해결을 위한 자동화된 데이터 처리에 집중함  
: ICT는 컴퓨팅을 포함하여 정보 처리 + 통신 기술 + 사회적 활용을 포괄하는 더 큰 개념  
: 실무에서는 두 용어가 구분없이 사용되나 학술적으로는 ICT가 더 넓은 범위를 다룸  


**ICT 발전 단계**
```
1단계 (전산화)
: 아날로그 정보를 디지털 데이터로 변환하여 처리함

2단계 (정보화)
: 웹, 이메일, 모바일을 통해 정보 전달과 공유가 폭발적으로 확대됨

3단계 (지능화)
: Cloud, Big Data, IoT 기술로 대규모 데이터의 수집, 저장, 분석이 가능해짐
: AI 기술의 실용화로 데이터 기반 의사결정이 시작됨
```

**기술 발전**

구분 | 3차 산업 혁명 | 4차 산업 혁명 | 5차 산업 혁명
---|---|---
시기 | 1960년대 ~ 2000년대 | 2010년대 ~ 진행중 | -
기술 영역 | ICT (정보통신기술) | ICT + AI 융합 | ICT + AI + 인간 협업
핵심 기술 | 컴퓨터, 인터넷, 디지털 | AI, IoT, Big Data | AI/Human Collaboration
사회 변화 | 정보 통신, 정보화 사회 | 초연결·초지능화 사회 | 인간 중심·지속가능 사회
기술 변화 | 정보화 기술 발전 | 디지털 기술들 융합 | 인간-기계 협업 강화
패러다임  | 아날로그에서 디지털 정보 처리로 전환 | 데이터 기반 의사 결정 | 인간 복지와 지속가능성 중심
생산 방식 | 디지털 자동화 시스템 | 지능형·자율형 시스템 | 인간-기계 협업 시스템
중심 가치 | 효율적 정보 처리 | 지능형 판단, 실시간 상호작용 | 인간 복지, 지속가능성, 회복력
