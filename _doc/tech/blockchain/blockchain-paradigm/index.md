# Blockchain Paradigm
: 분산 원장 기술을 기반으로 하는 새로운 사고방식과 접근 방법  
: 중개자 없는 P2P 네트워크, 암호학적 증명, 합의 알고리즘 등을 통해 새로운 형태의 협력과 가치 창출 방식을 제시함  
: 탈중앙화, 투명성, 신뢰성이라는 블록체인 핵심 가치 구현 및 혁신적인 접근 방식을 제공함
