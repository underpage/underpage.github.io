# Enterprise System
: 조직 수준의 비즈니스 목표를 달성하기 위해 구축된 대규모 통합 정보 시스템  
: 단순한 애플리케이션이 아니라 조직 전체의 업무 프로세스, 데이터, 인프라 등을 연결하는 복합적인 생태계  
: 다양한 기술 요소를 통합하여 목표 달성을 지원하고 지속적인 변화와 혁신을 가능하게 하는 기반을 제공함  
: 엔터프라이즈 시스템은 높은 가용성, 확장성, 신뢰성, 보안성을 필수적으로 요구함
