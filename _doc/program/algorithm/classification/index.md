# Algorithm Classification
: 알고리즘은 구현 방식, 설계 기법, 문제 도메인 등 다양한 기준으로 분류됨  
: 문제 해결에 따라 다양한 알고리즘이 존재하므로 체계화된 분류 체계가 존재하지 않음  
: 구체적인 알고리즘은 하나의 분류가 아니라 여러 분류에 속할 수 있음  
: 구체적인 알고리즘이란 개념 수준의 설계와 로직이 구현된 형태의 알고리즘을 의미함  


**용도**  
: 해결하고자 하는 문제의 성격에 따른 알고리즘  

- 기본 연산: 정렬, 탐색
- 구조 분석: 그래프, 트리 순회
- 최적화: 최단경로, 최적화, 근사
- 패턴 처리: 패턴 매칭, 파싱
