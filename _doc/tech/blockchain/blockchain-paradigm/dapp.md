# Decentralized Application
: 중앙 서버 없이 블록체인 네트워크 상에서 동작하는 애플리케이션  
: 핵심 기능(백엔드)이 스마트 컨트랙트로 구현되어 자동화된 방식으로 작동함  
: 완전한 탈중앙화 애플리케이션은 불가능하며 사용자 인터페이스는 일반적으로 중앙 서버에서 제공됨  
: 스마트 컨트랙트를 지원하는 플랫폼에서는 DApp을 개발 및 배포할 수 있음  


**구조**
```
┌─────────────────┐    ┌─────────────────┐    ┌─────────────────┐
│   Frontend UI   │ ←→ │   Web3 API/SDK  │ ←→ │ Smart Contract  │
└─────────────────┘    └─────────────────┘    └─────────────────┘
   사용자 인터페이스         블록체인 통신            비즈니스 로직
```
