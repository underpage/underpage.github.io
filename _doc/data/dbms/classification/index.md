# DBMS Classification
: 데이터베이스 관리 시스템은 데이터베이스 모델, 데이터 유형, 데이터 저장 방식 등 다양한 기준에 따라 분류됨  
: 일부 제품은 하나의 카테고리에 속하지 않고 여러 특성을 가지며 최근 제품은 범용적인 특성을 가짐  


**데이터베이스 모델**  
: [데이터베이스 모델](../../data-storage/database/database-model.md)을 기반으로 데이터 간 관계를 표현하고 접근하는 방식에 따른 분류

- HDBMS (계층형)
- NDBMS (네트워크형)
- RDBMS (관계형)
- OODBMS (객체지향)
- ORDBMS (객체관계)


**비정형 데이터**
- NoSQL
- NewSQL


**시스템 구조**  
: DBMS의 물리적 배치나 운영 아키텍처에 따른 분류

- DDBMS (분산)
- MMDBMS (메모리)
- VDBMS (벡터)
- SDBMS (공간)
