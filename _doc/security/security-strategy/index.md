# Security Strategy
: 보안 전략이란 조직의 자산을 보호하기 위한 포괄적인 계획을 의미함  
: 조직이 보호할 자산을 식별하고 이를 보호하기 위한 장기적인 청사진과 로드맵을 제시함  
: 보안 위협을 효과적으로 관리/대응하기 위한 방향성, 원칙, 우선 순위 등 실행 방법을 포함함

**보안 전략 목적**
- 위협으로부터 조직의 자산을 보호함
- 조직의 비즈니스 연속성을 보장함
- 보안 관련 법규를 준수함
- 보안을 통해 고객으로부터 신뢰를 확보함


**보안이 실패하는 이유**
- 기술적 요인
  - 시스템 복잡성 증가
  - 레거시 시스템 문제
  - 기술 변화 속도 : 기술 발전으로 기존 보안 대책 무력화
  - 보안 솔루션 간 통합 부족
- 관리적 요인
  - 보안 인식 부족
  - 예산, 인력 등 자원 제약
  - 정책은 존재하나 실제 적용되지 않음
- 인적 요인
  - 사용자 실수
  - 보안 정책 미준수
  - 내부자 위협
- 환경적 요인
  - 새로운 위협 등장
  - 공격 기술의 진화
  - 법률 및 규제 환경 변화
  - 새로운 기술
