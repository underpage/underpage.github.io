# Internet Computer Protocol
: 분산 애플리케이션을 구축, 실행, 배포할 수 있는 환경을 제공하는 플랫폼  
: 다른 플랫폼과 다르게 완전한 탈중앙화 컴퓨팅 플랫폼을 지향함  
: 전통적인 인터넷 서비스를 블록체인에서 구현할 수 있는 새로운 패러다임을 제시

- ICP 아키텍처
  - 네트워크 구조
  - 거버넌스 및 경제 모델
- ICP 핵심 기술
  - 캐니스터 모델
    - 캐니스터 생명주기
    - 캐니스터 상태 관리
  - 합의 메커니즘
  - Chain Key 암호화
  - 역방향 가스 모델
- ICP 개발 환경
- ICP 보안


**주요 특징**
- 전통적인 클라우드 컴퓨팅과 블록체인을 결합
- 플랫폼의 운영과 거버넌스에 사용되는 ICP 토큰 존재
- 캐니스터라는 단위를 사용해 스마트 컨트랙트 기능 제공
- 스마트 컨트랙트보다 더 광범위한 웹 서비스 기능 제공
