# JDK Version
: 약 6개월을 주기로 새로운 버전이 출시되며 장기 지원 버전이 아닌 경우 단기 지원을 받음  

**장기 지원 버전**  
: Java 11, 17, 21, 25 버전이 장기 지원 버전으로 약 5년간 지원됨 (벤더마다 차이 존재)  



## JDK 25
https://openjdk.org/projects/jdk/25

- Instance Main Methods: 메인 메소드 간략화


## JDK 24
https://openjdk.org/projects/jdk/24

- Stream Gatherers: 스트림 API 확장
- Post-Quantum Cryptography: 양자 내성 암호 알고리즘 일부 도입
- Primitive Types in Patterns (프리뷰): 패턴 매칭에서 기본 타입 지원
- Flexible Constructor Bodies (프리뷰): 생성자에서 super 호출 전 문장 허용


## JDK 23
https://openjdk.org/projects/jdk/23

- Markdown Documentation Comments: JavaDoc에서 마크다운 문법 지원
- Module Import Declarations (프리뷰): 모듈 전체 가져오기 지원


## JDK 22
https://openjdk.org/projects/jdk/22

- Locale-Dependent List Patterns
- Foreign Function & Memory API: JVM 외부 메모리 및 함수에 안전하게 접근하는 API


## JDK 21
https://openjdk.org/projects/jdk/21

- Record Patterns: 레코드 패턴 매칭
- String Templates (프리뷰): 문자열 템플릿


## JDK 18
https://openjdk.org/projects/jdk/18/

- UTF-8 by Default
- Internet-Address Resolution SPI
- Vector API
- Simple Web Server: 설정없는 간단한 웹 서버 지원 (테스트, 학습 용도)


## JDK 17
https://openjdk.org/projects/jdk/17

- Sealed Classes: Sealed 클래스 도입
- Enhanced Pseudo-Random Number Generators: 새로운 의사 난수 생성기 인터페이스를 도입


## JDK 16
https://openjdk.org/projects/jdk/16

- Records: 레코드 클래스 도입
- Packaging Tool


## JDK 15
https://openjdk.org/projects/jdk/15

- Text Blocks
- Hidden Classes
- Reimplement the Legacy DatagramSocket API
- Edwards-Curve Digital Signature Algorithm: 디지털 서명 알고리즘 EdDSA 도입


## JDK 14
https://openjdk.org/projects/jdk/14

- Switch Expressions


## JDK 12
https://openjdk.org/projects/jdk/12

- Unicode 11 Support
- JVM Constants API
- Microbenchmark Suite


## JDK 11
https://openjdk.org/projects/jdk/11/

- Local-Variable Syntax for Lambda Parameters
- Nest-Based Access Control
- HTTP Client
- Launch Single-File Source-Code Programs
- Transport Layer Security


## JDK 10
https://openjdk.org/projects/jdk/10

- Local-Variable Type Inference: 지역 변수 타입 추론
- Application Class-Data Sharing
- Thread-Local Handshakes
- Root Certificates 


## JDK 9
https://openjdk.org/projects/jdk9/

- Module System (Project Jigsaw)
- Diamond Operator enhancements
- try-with-resources Improvement
- SafeVarargs Annotation improvements
- Private methods in interfaces
- Factory methods for collections
- Stream API improvements
- JShell
- HTTP2 Support
