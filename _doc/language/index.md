# Programming Language
: 프로그래밍 언어의 목적, 용도, 특성 등에 따라 분류  
: 프로그래밍 언어는 하나의 특정 분류에만 속하지 않고 여러 분류에 동시에 포함될 수 있음  

**관련 용어**

용어 | 설명
---|---
런타임 | 프로그램이 실제로 실행되는 환경
플랫폼 | 특정 기술 위에서 애플리케이션이 실행될 수 있는 환경 (ex. 자바 플랫폼, .NET 플랫폼)
생태계 | 언어를 중심으로 형성된 개발 도구, 프레임워크, 커뮤니티 등 전체적인 환경

최근 프로그래밍 언어는 실험적인 언어가 아닌 이상 대부분 생태계를 형성함
