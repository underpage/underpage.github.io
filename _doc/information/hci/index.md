# Human Computer Interaction
: 인간과 컴퓨터 시스템 간 직관적이고 효율적인 상호작용을 연구하는 학문 및 기술 분야  
: 사용성, 접근성, 사용자 경험을 향상하기 위해 인지 심리학, 디자인, 컴퓨터 과학 등을 통합함  
: 인간의 인지적, 감정적, 사회적 특성을 고려하여 사용하기 쉽고 만족스러운 시스템을 설계하는 것을 목표로 함  
: 이를 위해 인간과 컴퓨터를 연결하는 인터페이스(상호작용이 일어나는 접점)를 설계하고 구현함  

```
인간 <> 인터페이스 <> 컴퓨터
```
